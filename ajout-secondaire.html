Ajout en Secondaire

Pour ajouter un nouveau membre en Secondaire

Ceci est valable uniquement si ce membre existe chez nous

  1. Cliquez sur le bouton Ajout en Secondaire

  2. Confirmez la recherche

    • Si la recherche est positive on peut utiliser cette fonction
    • Si la recherche est négative on ne peut pas utiliser cette fonction

  3. Vous obtenez une fiche remplie

  4. Vous remplissez la fiche suivant les besoins

    1. ActivitéS
    2. CPE + etc...
  5. N'oubliez pas le champ En cours

  6. Pour terminer vous cliquez sur :

    • Confirmation pour modifier ce membre
    • Annulation pour ne rien modifier
    • Aucun autre bouton ne peut être utilisé

  8. Vous cliquez en-dehors pour terminer votre action
