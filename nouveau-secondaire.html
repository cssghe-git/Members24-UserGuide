Nouveau en Secondaire

Pour ajouter un nouveau membre en Secondaire

Ceci est valable uniquement si ce membre n'existe pas chez nous

  1. Cliquez sur le bouton Nouveau en Secondaire

  2. Confirmez la recherche

    • Si la recherche est positive on ne peut pas utiliser cette fonction
    • Si la recherche est négative on peut utiliser cette fonction

  3. Vous obtenez une fiche vide

  4. Vous remplissez la fiche suivant les besoins

    1. Nom-Prénom
    2. CDC
    3. ActivitéS
    4. CPE + etc...
  5. N'oubliez pas le champ En cours

  6. Pour terminer vous cliquez sur :

    • Ajout nouveau membre pour créer ce membre
    • Aucun autre bouton ne peut être utilisé

  8. Vous cliquez en-dehors pour terminer votre action
