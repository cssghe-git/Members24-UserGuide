Introduction

Cette fonctionnalité vous permet de connaître les différentes valeurs des champs pour la gestion de votre liste via Notion.

Les différentes valeurs des champs

Statuts
Valeur Description Champs obligatoires
Nouveau en principal Ajout d'un nouveau membre CDC - ActPrc
Nouveau en secondaire Ajout d'un nouveau membre d'un autre CDC CDC - ActSecs
Ajout en secondaire Ajout d'un membre existant dans l'activité secondaire ActSecs
Modification Modificatio d'un ou plusieurs champs
Suppression Suppression d'un membre dans une activité ActPrc (si suppression en principal) - ActSecs (si suppression en secondaire)
Arrêt Le membre ne vient plus (pour l'instat ? ) Date sortie
Décès Le membre est décédé Date décès
CPE
Valeur Description
Cotisant si paiement dans notre CDC
Participant si paiement dans un CDC du BW
Extérieur si paiement en dehors du BW
Cotisation
Valeur Description
0 cotisation pas encore payée
17 cotisation payée avant le 01/09
9 cotisation payée à partir du 01/09
Eneo-EneoSport
Valeur Description
Eneo si activité non sportive
EneoSport si activité sportive
V-A
Valeur Description
V si volontaire dans votre activité
A si animateur dans votre,activité
VA si volontaire et animateur
