Nouveau en Principal

Pour ajouter un nouveau membre en Principal

Ceci est valable uniquement si ce membre n'existe pas encore chez nous

  1. Cliquez sur le bouton Nouveau en Principal

  2. Confirmez la recherche

    • Si la recherche est positive on ne peut pas utiliser cette fonction, seule une Modification est plausible
    • Si la recherche est négative on cinfirme que l'on continue

  3. Vous obtenez une fiche vide

  4. Vous remplissez la fiche suivant les besoins

    1. Nom-Prénom
    2. CDC
    3. ActivitéP
    4. Civilité + Adresse + Code postal + Localité
    5. GSM + Téléphone + Mail
    6. Date de naissance
    7. CPE + etc...
  5. N'oubliez pas le champ En cours

  6. Pour terminer vous cliquez sur :

    • Ajout nouveu membre pour créer ce membre
    • Aucun autre bouton ne peut être utilisé

  8. Vous cliquez en-dehors pour terminer votre action
